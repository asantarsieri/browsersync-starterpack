# browsersync-starterpack

Simple [browsersync](https://www.browsersync.io) starterpack with html and css filewatcher.
### Contains

 - Gulp
 - Browsersync

### Usage

 - `$ npm install`
 - `$ sudo npm install gulp -g`
 - run `$ gulp` or `$ gulp serve`
 
### dev dependencies
 - Node.js
 - npm
 - browser-sync:  ^2.18.12
 - gulp: ^3.9.1
